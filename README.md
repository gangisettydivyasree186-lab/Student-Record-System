# Student Record Management System

Java CLI-based CRUD application to manage student records.

## Features
- Add student record
- View all students
- Update student details
- Delete student record

## Technologies Used
- Java
- ArrayList (Collections)
- OOP concepts
- Command Line Interface

## How to Run

Compile:
javac Student.java StudentManager.java

Run:
java StudentManager
